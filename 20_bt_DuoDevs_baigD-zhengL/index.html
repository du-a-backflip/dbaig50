Alerts

There are six kinds of alerts: primary, secondary, success, danger, warning, info, light, and dark!

Format in html:
<div class = "alert alert-[ALERT TYPE]" role = "alert">
TEXT YOU WANT TO PUT IN ALERT </div>

You can put things to dismiss the alerts within the div tags but you must add alert-dismissable after div class to do so. You can use links or buttons to dismiss the alert.

For links:
<a href="#" class="close" data-dismiss="alert" aria-label="close">×</a>

For buttons:
<button type="button" class="close" data-dismiss="alert" aria-label="Close">
<span aria-hidden="true"> ×</span>
</button>
